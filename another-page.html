What is DOM? What is the purpose of DOM+

DOM means Document Object Model created by the browser when a web page is loaded. In graphical form, it looks like a tree of elements/nodes. Purpose: As DOM is a programming interface, it allows to create, change, or remove elements from a website document. DOM manipulation is when use JavaScript to add, remove, and modify elements of a website

How to select element from DOM?

  • getElementById: The selection is based on the Id name.This selector returns only the first matched element while ignoring the remaining.
  • getElementsByTagName This method returns all the elements that matches the specified Tag name.
  • getElementsByClassName: This method returns all the elements that matches the specified Class name.

    • What is Event bubbling?+

    Event bubbling is a process that starts with the element that triggered the event and then bubbles up to the containing elements in the hierarchy.The event is first captured and handled by the innermost element and then propagated to outer elements.

    What is call back function in javascript? What does it do?+

    A callback function is a function that is passed as an argument to another function so that it can be executed in that other function.
